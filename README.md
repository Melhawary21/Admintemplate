# AegisBoard Admin Template

A premium Bootstrap 5 HTML admin dashboard template package designed for ThemeForest-quality commercial use.

## Included
- 60+ HTML pages (dashboards, apps, components, forms, tables, charts, auth, system pages)
- SCSS architecture with modular partials
- Vanilla JS architecture with theme customizer and sidebar control
- Documentation starter, changelog, credits, support policy, and ThemeForest listing draft

## Quick Start
Open `html/index.html` in your browser.
