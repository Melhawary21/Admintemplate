# ThemeForest Listing Draft

## Product Description
AegisBoard is a premium, clean, and highly reusable Bootstrap 5 admin dashboard template crafted for SaaS, CRM, ERP, analytics, finance, HR, and property management products. It combines enterprise-grade UX with developer-friendly architecture, giving teams a strong foundation for commercial dashboards and internal applications.

## Feature List
- 60+ production-ready HTML pages
- Multiple dashboards: Executive, Analytics, Sales, CRM, Projects, Property Management
- Layout variants: vertical, horizontal, collapsed, boxed, dark, RTL
- Built-in app pages: Chat, Email, Kanban, Calendar, Tasks, Invoice, Files, Contacts
- Rich UI components, forms, tables, and chart pages
- Light/Dark mode + LTR/RTL + compact/comfortable + boxed/full settings customizer
- Bootstrap 5 + SCSS modular architecture + Vanilla JavaScript modules
- Accessibility-conscious semantics and keyboard-friendly interactions
- Clean, maintainable naming conventions (`.ab-*`, `data-ab-*`)
- Buyer-friendly documentation and structured source files

## Package Structure Explanation
- `/html/`: all page templates
- `/assets/scss/`: source styles organized by concern
- `/assets/css/`: distributable compiled styles
- `/assets/js/`: modular JS architecture
- `/partials/`: reusable fragments for static build workflows
- `/documentation/`: setup, customization, changelog, credits, support policy
