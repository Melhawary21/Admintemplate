# Getting Started

## Project Structure
- `/html/` — production HTML templates
- `/assets/scss/` — modular SCSS source
- `/assets/css/` — compiled distribution styles
- `/assets/js/` — runtime JS modules
- `/documentation/` — buyer docs and support assets

## Setup
1. Open `html/index.html` for the default dashboard.
2. Compile `assets/scss/app.scss` into `assets/css/app.css` for custom theme builds.
3. Use customizer controls to test `light/dark`, `ltr/rtl`, `comfortable/compact`, and `fluid/boxed` states.

## Accessibility Baseline
- Skip link included for keyboard users.
- Visible focus states on all form controls and interactive elements.
- Semantic landmarks for header/navigation/main content.
