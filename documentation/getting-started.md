# Getting Started
1. Open `/html/index.html` to preview the default dashboard.
2. Compile `assets/scss/app.scss` to `assets/css/app.css` for production customization.
3. Use `data-theme`, `dir`, `data-density`, and `data-ab-width` attributes on `<html>` for runtime layout modes.
4. Keep shared patterns consistent with `.ab-*` naming strategy.
