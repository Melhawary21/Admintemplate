# Customization Guide
- Edit `assets/scss/_variables.scss` for colors, radius, spacing, shadows.
- Use `assets/css/app-dark.css` and `assets/css/app-rtl.css` for dark and RTL bundles.
- Update sidebar/menu structure across pages or integrate partials with your static build tool.
- Theme customizer behavior lives in `assets/js/theme-customizer.js`.
